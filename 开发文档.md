# 打印中转程序开发文档

## 1. 项目介绍

打印中转程序是一个基于 Electron + Vue3 + TypeScript 开发的 Windows 平台应用，主要功能包括：

### 核心功能

- **打印机扫描**：自动扫描本机所在网段，检测开放 9100 端口的网络打印机
- **打印机管理**：查看打印机状态、名称、IP 地址等信息
- **默认打印机设置**：设置全局唯一默认打印机
- **WebSocket 服务**：监听 13555 端口，接收打印任务并转发到默认打印机
- **打印历史记录**：记录所有打印任务的状态和结果

### 技术架构

```
┌─────────────────────────────────────────────────────────────┐
│                      Electron 主进程                         │

│  │  PrinterScanner  │  │  PrintService    │  │WebSocket │ │
│  │  (打印机扫描)     │  │  (打印服务)      │  │ Server    │ │
│  └──────────────────┘  └──────────────────┘  └───────────┘ │
└─────────────────────────────────────────────────────────────┘
                           │ IPC

┌─────────────────────────────────────────────────────────────┐
│                      Vue3 渲染进程                           │
│  ┌──────────┐  ┌────────────┐  ┌──────────────────────┐    │
│  │PrinterTabs│  │PrinterInfo │  │ PrintHistory         │    │
│  │ (打印机Tab)│  │ (打印机信息)│  │ (打印历史记录)       │    │
│  └──────────┘  └────────────┘  └──────────────────────┘    │
└─────────────────────────────────────────────────────────────┘
```

## 6. 使用说明

### 6.1 扫描打印机

1. 启动应用后，点击「重新扫描」按钮
2. 应用会自动扫描本机所在网段（如 192.168.1.x）
3. 扫描完成后，会显示所有开放 9100 端口的打印机

**手动扫描**：在输入框中输入自定义网段（如 `192.168.0`），然后点击扫描

### 6.2 设置默认打印机

1. 在打印机 Tab 中选择目标打印机
2. 点击「设为默认」按钮
3. 设置成功后，该打印机会显示「默认」标识

### 6.3 WebSocket 接口调用示例

#### 连接地址
```
ws://localhost:13555
```

#### 请求格式
```json
{
  "printUrl": ["https://example.com/document.pdf", "https://example.com/file2.pdf"]
}
```

#### 成功响应
```json
{
  "code": 0,
  "msg": "打印成功"
}
```

#### 失败响应
```json
{
  "code": -1,
  "msg": "失败原因"
}
```

#### JavaScript 调用示例
```javascript
const ws = new WebSocket('ws://localhost:13555')

ws.onopen = () => {
  ws.send(JSON.stringify({
    printUrl: ['https://example.com/test.pdf']
  }))
}

ws.onmessage = (event) => {
  const response = JSON.parse(event.data)
  console.log(response)
}
```

### 6.4 打印状态说明

| 状态 | 说明 |
|------|------|
| 发送中 | PDF 正在下载或正在发送到打印机 |
| 成功 | 打印任务完成 |
| 失败 | 网络异常、打印机离线或其他错误 |

## 7. 常见问题

### 7.1 端口占用

**问题**：WebSocket 服务无法启动，提示端口已占用

**解决**：
1. 检查 13555 端口是否被其他程序占用
2. 使用以下命令查看端口占用：
   ```bash
   netstat -ano | findstr "13555"
   ```
3. 关闭占用端口的程序，或重启电脑

### 7.2 打印机离线

**问题**：扫描到打印机，但状态显示离线

**解决**：
1. 检查打印机是否正常连接到网络
2. 确认打印机 IP 地址正确
3. 尝试在浏览器中访问打印机 Web 界面
4. 检查防火墙是否阻止了 9100 端口

### 7.3 打印失败

**问题**：打印任务失败

**可能原因**：
1. 默认打印机未设置
2. 打印机离线或网络断开
3. PDF URL 无法访问或下载超时
4. 打印机不支持直接接收 PDF 数据

**排查方法**：
1. 检查默认打印机设置
2. 确认打印机状态为在线
3. 验证 PDF URL 可正常访问
4. 查看打印历史中的错误信息

### 7.4 扫描不到打印机

**问题**：扫描完成后没有显示任何打印机

**解决**：
1. 确认打印机已开启并连接到同一网络
2. 检查打印机是否支持 9100 端口（RAW 打印协议）
3. 尝试手动输入网段进行扫描
4. 检查防火墙是否阻止了端口扫描

## 8. 项目结构

```
src/
├── main/                    # 主进程代码
│   ├── index.ts             # Electron 入口
│   ├── preload.ts           # 预加载脚本
│   ├── services/            # 服务类
│   │   ├── printerScanner.ts # 打印机扫描服务
│   │   ├── printService.ts   # 打印服务
│   │   └── webSocketServer.ts # WebSocket 服务
│   ├── types/               # 类型定义
│   │   └── index.ts
│   └── utils/               # 工具函数
│       ├── network.ts        # 网络工具
│       └── storage.ts        # 本地存储工具
└── renderer/                # 渲染进程代码
    ├── App.vue              # 根组件
    ├── main.ts              # Vue 入口
    ├── components/          # 组件
    │   ├── PrinterTabs.vue   # 打印机 Tab 组件
    │   ├── PrinterInfo.vue   # 打印机信息组件
    │   └── PrintHistory.vue  # 打印历史组件
    ├── composables/         # 组合式函数
    │   └── usePrinter.ts     # 打印机相关逻辑
    ├── types/               # 类型定义
    │   └── index.ts
    └── utils/               # 工具函数
        └── format.ts         # 格式化工具
```

## 9. 配置说明

### 9.1 端口配置

- **WebSocket 端口**：13555（固定）
- **打印机端口**：9100（固定，网络打印机标准端口）

### 9.2 数据存储

应用数据存储在以下位置：
```
%APPDATA%\print-relay-app\
├── printers.json    # 打印机列表
├── history.json     # 打印历史
└── settings.json    # 设置（默认打印机ID）
```
